POS Product Ordering

Way to order products in POS

No Configuration

No need for configuration, just the installation of the module.

POS Interface

There are 5 ways to order the products in POS interface

Default - The default behavior of product ordering in POS. Usually, it is ordered by Sequence, Internal Reference and Product Name.

A → Z - Order by ascending product name alphabetical order.

Z → A - Order by descending product name.

$ → $$$ - Order by ascending product price.

$$$ → $ - Order by descending product price.

Compatibility

This module is compatible and tested with these modules:

  • ☑   Restaurant module (pos_restaurant)
  • ☑   POS Discount module (pos_discount)
  • ☑   Point of Sale / Product Limit
